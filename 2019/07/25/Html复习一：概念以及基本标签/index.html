1. Html基本信息

1.1 Html 描述

HTML , 英文全称为HyperText Markup Language , 也就是超文本标记语言，在网络的高速发展过程中，人们对于浏览器的显示网页的内容不再满足于单纯的文本信息，Html则应运而生。

1.2 Html 规范

作为一门语言，与其他语言一样，html也有一套自己的规范，基本如下

  1. 每个html文件都必须以 <html>开始 ，以 </html> 结束。

  2. <head> 标签负责再初始化页面的相关信息（如标题栏）。

  3. <body> 标签负责显示页面内容，页面内容页必须写在该标签下。

  4. 一般来说，html里的标签有开始就一定要结束，例如开始时<head> ，就一定要有</head> 来结束。

    某些标签，例如换行，没有结束标签，但是html也为这些标签规定了一个规范，将/ 放在标签名后，表示在标签内结束，如换行标签<br/>

  1. html的代码不区分大小写，也就是说，你写<head> 和 <Head> 并不会出现问题，当然，Java是严格区分大小写的。

1.3 Html 的操作思想。

网页中有很多数据，不同的数据可能需要不同的显示效果，这个时候需要使用标签把要操作的数据包起来（封装起来），通过秀爱标签的属性值实现标签内数据样式的变化。
